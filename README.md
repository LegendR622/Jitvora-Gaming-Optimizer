# Redline Gaming Optimizer

**Redline** is a Windows desktop app that helps PC gamers **clean up**, **tune**, and **check** their system in one place — with real measurements (CPU, RAM, ping, drivers) and clear actions, not guesswork.

This repository hosts **official installers only**. Source code is not published here.

---

## What Redline does

| Area | What you get |
|------|----------------|
| **Performance** | Gaming readiness score, power plan & Game Mode checks, FPS-oriented tweaks |
| **Cleaner** | Safe scan of cache/temp folders; optional browser history & cookies (browsers are closed first) |
| **Drivers** | Hardware scan, installed driver versions, status, vendor links & guided updates (winget) |
| **Network** | Ping test, DNS comparison, speed test, adapter overview |
| **Security & repair** | Quick security check, SFC/DISM, Winsock reset, restore points |
| **AI Advisor** | Answers *“Why FPS drops?”*, *“Why is ping bad?”*, *“What slows my PC?”* from **live system data** (local rules, no cloud) |
| **Game profiles** | Tips for games like Rust & ARC Raiders (settings, known issues, anti-stutter hints) |
| **Updates** | Built-in check for new versions from this GitHub release |

## Privacy & safety

- Advisor runs **on your PC only** — no chat data sent to the cloud for analysis
- Cleaner targets **known cache/temp locations**; destructive options (history/cookies) are **opt-in**
- Driver updates use **official sources** (vendor pages / winget)
