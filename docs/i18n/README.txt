Website i18n tools (run from repo root or this folder):

  python docs/i18n/generate-i18n.py   → regenerates docs/site-i18n.js
  python docs/i18n/audit-i18n.py      → checks 100% locale coverage
  python docs/i18n/patch-i18n.py      → patch English-identical strings
  python docs/i18n/add-hero-mock-keys.py  → one-off key helper (reference)

Source of truth: docs/i18n-full.json
